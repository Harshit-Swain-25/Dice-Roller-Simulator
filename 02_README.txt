==============================
    Dice Roller Simulator
==============================

Description:
A Python program that simulates rolling a dice.
The user can roll once or multiple times, and
the result will be random between 1 and 6.

Features:
- Simulates a six-sided dice
- Uses random number generation
- Allows repeated rolling
- Exits when user chooses

How to Run:
1. Save the file as 01_main.py
2. Open VS Code terminal.
3. Run the program with:
   python3 01_main.py
