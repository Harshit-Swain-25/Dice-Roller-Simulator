# Dice Roller Simulator using Python

A command-line Dice Roller Simulator built using Python. This project simulates rolling one or multiple dice and allows users to customize both the number of dice and the number of sides on each die.

The application generates random outcomes and displays the roll results, total score, and occurrence count for each value.

## 📌 Project Overview

This project is designed to replicate real-world dice rolling in a digital environment.

Users can choose:

* The number of sides on each die
* The number of dice to roll

The program then generates random results and summarizes the outcomes in an easy-to-read format.

This project is suitable for beginners who want to practice Python programming and learn how random number generation works.

## 🚀 Features

* Roll one or multiple dice
* Customize the number of sides
* Display individual roll results
* Calculate total score
* Display frequency counts
* Interactive command-line interface
* Repeat roll functionality
* Beginner-friendly implementation

## 🛠️ Technologies Used

## 🎲 How It Works

1. Choose the number of sides for each die.
2. Choose how many dice to roll.
3. The program generates random results.
4. Individual roll values are displayed.
5. If multiple dice are rolled, the total score and frequency counts are shown.
6. Users can continue rolling or exit the program.
